# 해볼 것들

* 독립적인 프로세스로 띄우는 것을 옵셔널로 바꾸고, 기본적으로 차일드 프로세스로 Life Cycle을 관리할 수 있게 바꾸기.
* 차일드 프로세스를 실행 중인지 모니터링 하기.
* 죽으면 다시 띄우는 기능 추가.
* 한 개 뿐만 아니라, 여러 개도 띄울 수 있게 하기.
* Socket dup을 통한 TCP Proxy 기능 추가하기 (using Tokio)
